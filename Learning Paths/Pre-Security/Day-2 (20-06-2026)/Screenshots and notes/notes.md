## DNS in Detail

### Key Concepts Learned

* DNS (Domain Name System) translates domain names into IP addresses.
* DNS allows users to access websites using human-readable names instead of numerical IP addresses.
* DNS queries follow a hierarchy:

  * Local Cache
  * Recursive Resolver
  * Root DNS Server
  * TLD Server
  * Authoritative DNS Server

### Important DNS Record Types

* A Record – Maps a domain to an IPv4 address.
* AAAA Record – Maps a domain to an IPv6 address.
* CNAME Record – Alias for another domain.
* MX Record – Handles email routing.
* TXT Record – Stores text information for verification and security.

### Key Takeaways

* DNS is one of the fundamental services of the Internet.
* Every website visit usually begins with a DNS lookup.
* Understanding DNS helps in troubleshooting networking and cybersecurity issues.

## HTTP in Detail

### Key Concepts Learned

* HTTP (HyperText Transfer Protocol) is used for communication between clients and web servers.
* HTTPS is the secure version of HTTP that uses encryption through TLS/SSL.

### Common HTTP Methods

* GET – Retrieve data from a server.
* POST – Submit data to a server.
* PUT – Update existing data.
* DELETE – Remove data from a server.

### HTTP Components

* Request Line
* Headers
* Body (optional)

### Response Components

* Status Codes
* Headers
* Response Body

### Common Status Codes

* 200 OK
* 301 Moved Permanently
* 403 Forbidden
* 404 Not Found
* 500 Internal Server Error

### Practical Skills

* Sent GET, POST, PUT, and DELETE requests.
* Worked with URL and body parameters.
* Observed HTTP request and response structures.

### Key Takeaways

* HTTP is the foundation of web communication.
* Understanding requests and responses is essential for web security testing and cybersecurity.

## What is Networking?

### Key Concepts Learned

* A network is a collection of devices connected to exchange information.
* Devices communicate using IP addresses and networking protocols.

### Important Networking Concepts

* IP Address
* Router
* Switch
* Packet
* Protocol
* Port

### Common Protocols

* HTTP/HTTPS
* DNS
* SSH
* FTP

### Key Takeaways

* Networking is the foundation of cybersecurity.
* Understanding how data travels between devices is critical for future penetration testing and security analysis.
