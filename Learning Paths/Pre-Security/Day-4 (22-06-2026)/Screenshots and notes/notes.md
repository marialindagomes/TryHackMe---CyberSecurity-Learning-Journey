## Inside a Computer System

### What I Learned

* A computer system is made up of several hardware components working together.
* The CPU executes instructions and processes data.
* RAM temporarily stores data that is actively being used.
* Storage devices such as SSDs and HDDs permanently store data.
* The motherboard connects all hardware components.
* Input and output devices allow users to interact with computers.

### Key Components

| Component    | Function                  |
| ------------ | ------------------------- |
| CPU          | Executes instructions     |
| RAM          | Temporary memory          |
| SSD/HDD      | Permanent storage         |
| Motherboard  | Connects hardware         |
| GPU          | Graphics processing       |
| Power Supply | Supplies electrical power |

### Key Takeaways

* Every hardware component performs a specific role.
* Understanding hardware helps with troubleshooting and system administration.
* Hardware knowledge is important for cybersecurity professionals.

## Virtualisation Basics

### What I Learned

* Virtualisation allows multiple operating systems to run on a single physical machine.
* Virtual Machines (VMs) operate independently from one another.
* A hypervisor manages virtual machines and allocates system resources.
* Virtualisation improves resource utilization and reduces hardware costs.
* Snapshots allow virtual machines to be restored to previous states.

### Important Terms

#### Host Machine

The physical computer running the virtualization software.

#### Guest Machine

A virtual machine running inside the host system.

#### Hypervisor

Software that creates and manages virtual machines.

Examples:

* VirtualBox
* VMware
* Hyper-V

### Advantages of Virtualisation

* Better resource utilization
* Safe testing environments
* Isolation between systems
* Reduced hardware costs
* Easy deployment and recovery

### Cybersecurity Relevance

* Provides safe environments for penetration testing.
* Allows malware analysis without affecting the host system.
* Enables multiple operating systems to run simultaneously.
* Forms the foundation of many cybersecurity labs, including TryHackMe.
