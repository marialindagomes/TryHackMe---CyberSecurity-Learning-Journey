## Offensive Security Intro

### What I Learned

* Offensive Security involves testing systems by thinking like an attacker.
* Ethical hackers identify vulnerabilities before malicious attackers can exploit them.
* Penetration testing is used to assess the security of applications and systems.
* Offensive security professionals are commonly known as Red Team members.

### Key Terms

* Ethical Hacking
* Penetration Testing
* Vulnerability Assessment
* Red Team

---

## Defensive Security Intro

### What I Learned

* Defensive Security focuses on monitoring, detecting, and protecting systems.
* Security professionals analyze suspicious activities and respond to threats.
* Monitoring tools help identify unusual behavior in networks and devices.
* Security Operations Centers (SOC) play a key role in defending organizations.

### Key Terms

* SOC (Security Operations Center)
* SIEM (Security Information and Event Management)
* Threat Intelligence
* DFIR (Digital Forensics and Incident Response)
* Malware Analysis

## Careers in Cyber

### Cybersecurity Roles Explored

* SOC Analyst
* Penetration Tester
* Security Engineer
* Incident Responder
* Threat Hunter
* Malware Analyst

### Key Takeaway

Cybersecurity offers multiple career paths, including offensive security, defensive security, cloud security, digital forensics, and security engineering.
